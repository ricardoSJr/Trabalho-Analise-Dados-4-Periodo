## Sobre o Projeto
Este trabalho acadêmico tem como objetivo analisar os fatores socioeconômicos, estruturais e de desempenho associados à evasão escolar no Ensino Médio na região da Grande Vitória (Vitória, Cariacica, Vila Velha e Serra), utilizando microdados do INEP.

O trabalho foi dividido em 3 partes:
- [x] **Parte 1:** Contextualização, Objetivo e Limpeza de Dados.
- [x] **Parte 2:** Cruzamento de Microdados, Taxas de Abandono e Geração de Gráficos (Insights Iniciais).
- [x] **Parte 3:** Modelagem Preditiva com Random Forest e Conclusão.
